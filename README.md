# Currency Converter

A simple and responsive Currency Converter built using **HTML, CSS, and JavaScript**. It uses a live Currency API to convert between different currencies and updates country flags automatically.

## 🚀 Features

* Live currency conversion
* Multiple currency support
* Automatic flag updates
* Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Currency API
